# London Bike Sharing Analytics & Visualization

## Project Overview

This project analyzes the London Bike Sharing dataset to explore usage patterns, environmental impacts, and user behavior trends using Python and Tableau.

The project combines data preprocessing, exploratory data analysis (EDA), interactive dashboarding, and storytelling visualization techniques.

---

## Objectives

- Analyze bike-sharing demand trends
- Understand how weather conditions affect bike rentals
- Explore hourly and seasonal riding patterns
- Create interactive Tableau dashboards
- Support data-driven transportation planning

---

## Dataset Information

The dataset contains more than 17,000 records of bike-sharing activity in London between 2015 and 2017.

### Features Include:
- Timestamp
- Bike Rental Count
- Temperature
- Humidity
- Wind Speed
- Weather Conditions
- Holiday Indicator
- Weekend Indicator
- Season

---

## Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Tableau
- Jupyter Notebook

---

## Key Insights

- Bike usage peaks around:
  - 7 AM
  - 5 PM

- Clear weather significantly increases bike usage.

- Rain, snow, and strong wind reduce rental activity.

- Moderate temperatures produce the highest bike demand.
